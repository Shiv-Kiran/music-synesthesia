# Qualia (music-synesthesia)

Local development repository for Qualia, a mic-driven synesthetic music visualizer.

## Scripts

- `pnpm dev` - start the Next.js app
- `pnpm build` - production build
- `pnpm lint` - run ESLint
- `pnpm typecheck` - run TypeScript checks
- `pnpm test` - run Vitest test suite

## Notes

- `Plan.md` and `Tasks.md` are local planning documents and are intentionally ignored by git.
- `Qualia.pdf` contains the source design/spec discussion used to plan Milestone A.
